Browser Fingerprinting Census

Browser Fingerprinting is a web tracking technique often used to create profiles and stories about you and millions of other users. We have audited [[5000]] of the most visited websites to shine a light on the practice of fingerprinting. Here are our some of our results.

What is Online Fingerprinting?

A form of user tracking.

Browser fingerprinting is a tracking technique executed by running a code that requests information from a user's browser, and based on that information, creates a unique fingerprint for that particular user.

What is done with users' fingerprints?

This information is sometimes used to personalize a website or service user experience, but is also used to create profiles and stories about millions of users.

Audio fingerprinting?! Canvas fingerprinting?!

There are several kinds of browser fingerprinting techniques. Known techniques include font, canvas, webRTC, and audio fingerprinting. Though they vary in methodology, the goal is the same: mainly to uniquely identify a user and keep track of her movement online.

About

About

[[ ]] is a collaboration between the Data Transparency Lab ("DTL") , and DTL grantees Steven Englehardt and Arvind Narayanan of Princeton University , the researchers behind the Princeton Web Census.

The Princeton Web Census is a research tool that provides information about the measurements performed on 1 million web sites. The measurements were focused on identifying the tracking techniques used by those websites, both stateful (cookie-based) and stateless (fingerprinting-based). The tool can be used to check which tracking techniques are used by a given website or to check how widely used a given tracking technology is.

This website is an effort to interface, communicate, and share the Web Census data, which has been enhanced by Privacymeter, a tool created by the DTL Tech Program . You can find other privacy tools supported by DTL here . At the moment, the website is only visualizing data highlights related to fingerprinting, and not the other tracking techniques identified by the Web Census data. We are working on visualizing that data on this website soon.

Methodology

The Princeton Web Census is made possible by their open-source web privacy measurement tool, OpenWPM1 , which uses an automated version of a full-fledged consumer browser. It supports parallelism for speed and scale, automatic recovery from failures of the underlying browser, and comprehensive browser instrumentation.

Privacymeter builds on top of the OpenWPM1 results (the analysis of 1,000,000 sites) and hones in on the websites with the most traffic (as ranked by Alexa ), analyzing over 80,000 websites, which encompass 145 countries and 17 website categories.
