About

About

[[ ]] is a collaboration between the Data Transparency Lab ("DTL") , and DTL grantees Steven Englehardt and Arvind Narayanan of Princeton University , the researchers behind the Princeton Web Census.

The Princeton Web Census is a research tool that provides information about the measurements performed on 1 million web sites. The measurements were focused on identifying the tracking techniques used by those websites, both stateful (cookie-based) and stateless (fingerprinting-based). The tool can be used to check which tracking techniques are used by a given website or to check how widely used a given tracking technology is.

This website is an effort to interface, communicate, and share the Web Census data, which has been enhanced by Privacymeter, a tool created by the DTL Tech Program . You can find other privacy tools supported by DTL here .

Methodology

The Princeton Web Census is made possible by their open-source web privacy measurement tool, OpenWPM1 , which uses an automated version of a full-fledged consumer browser. It supports parallelism for speed and scale, automatic recovery from failures of the underlying browser, and comprehensive browser instrumentation.

Privacymeter builds on top of the OpenWPM1 results and hones in on the websites with the most traffic, analyzing over 80,000 websites, which encompass 145 countries and 17 website categories.
